## 📌 Featured Projects

### 🚀 CyberCorp Lab — Enterprise Pentesting & Detection Simulation

A realistic enterprise cybersecurity lab designed to simulate attack and defense workflows across segmented networks.

- Built a segmented lab environment with **pfSense, Active Directory, VPN, DMZ, vulnerable applications, Windows/Linux systems, and monitoring tools**
- Simulated attack paths involving **phishing, VPN access, SQL injection, privilege escalation, lateral movement, and Active Directory compromise**
- Used **Security Onion, Suricata, Sysmon, Wireshark, and firewall logs** to monitor, detect, and investigate malicious activity
- Documented findings, remediation steps, and investigation notes in a report-style format

---

### 🛡️ Application Security Testing Lab

Hands-on web application security project focused on OWASP Top 10 testing and vulnerability validation.

- Tested vulnerabilities such as **SQL Injection, XSS, CSRF, SSRF, IDOR, insecure file upload, and broken access control**
- Used **Burp Suite, OWASP ZAP, SQLMap, and manual testing techniques**
- Created vulnerability reports with **risk rating, impact, evidence, and remediation guidance**
- Practiced secure SDLC concepts, remediation validation, and retesting workflows

---

### 🔐 IAM & SSO Security Project

Identity-focused project based on enterprise IAM concepts and secure authentication workflows.

- Designed IAM workflows around **SSO, SAML, RBAC, LDAP, Active Directory, and access control**
- Practiced authentication and authorization testing across application flows
- Documented common IAM risks such as **misconfigured SSO, weak access controls, excessive privileges, and federation issues**
- Mapped IAM concepts to real-world enterprise security operations
